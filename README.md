## Project Overview
This project is a website for *Zakwantu Projects and Services (ZPS)*, an Eastern Cape-based organisation that focuses on community development, education, and awareness programmes. The website serves as an information hub for the organisation, providing details on services, community initiatives, donation options, and ways to get involved (volunteering, bursaries, sponsorships).  

The website is built using *HTML5, CSS3, and JavaScript*, following best practices for usability, accessibility, and responsive design.

---

## Website Goals and Objectives
- Provide a professional online presence for ZPS.  
- Showcase the organisation’s services, objectives, team, and projects.  
- Allow users to submit enquiries, applications, and donations via forms.  
- Improve user engagement through interactive elements, including maps and galleries.  
- Implement SEO best practices to increase visibility.  

---

## Key Features and Functionality
- *Responsive Design:* Mobile-first and works across devices.  
- *Navigation Menu:* Consistent across all pages with clear links.  
- *Forms with Validation:*  
  - Enquiry form for bursaries, volunteering, and sponsorships.  
  - Contact form with email integration and client-side validation.  
- *Interactive Map:* Shows locations relevant to ZPS activities using Leaflet.js.  
- *Image Gallery:* Lightbox-enabled for viewing photos.  
- *SEO Optimisation:*  
  - Meta titles and descriptions for all pages.  
  - Keyword research integrated in page content.  
  - Clean URL structure and internal linking.  
  - Sitemap (sitemap.xml) and robots.txt included.  
- *Social Media Integration:* Links to ZPS social media profiles (but they dont have social media accounts yet) 
- *Page Speed & Security:* Optimised for faster loading times and basic security best practices.
